# **App Name**: Simple Sheet

## Core Features:

- Text Input: Allow the user to input and edit text directly on the sheet.
- Formatting Options: Provide basic text formatting options (bold, italics, underline).
- Generate sheet name: Use AI tool to name sheet appropriately.

## Style Guidelines:

- Primary color: Light, desaturated blue (#B0C4DE) to create a calm and focused atmosphere, reminiscent of traditional paper.
- Background color: A clean, off-white (#F5F5F5) to emulate a blank sheet of paper and minimize distractions.
- Accent color: A slightly darker shade of blue (#778899) to highlight active elements and selections, ensuring clarity without being intrusive.
- Body and headline font: 'Inter', a grotesque-style sans-serif, which offers a modern and neutral appearance for both headlines and body text.
- Use simple, monochrome icons for formatting options.
- A clean, minimalist layout with a focus on maximizing the usable space for the text area.
- Subtle animations for UI interactions, such as fading in and out text.
