
# Описание файлов проекта

Это простое Next.js приложение. Ниже приведено описание каждого файла в проекте.

## Файлы в корневом каталоге

*   `package.json`: Этот файл содержит список зависимостей проекта и скрипты для запуска.
*   `package-lock.json`: Этот файл гарантирует, что каждый раз устанавливаются одни и те же зависимости.
*   `postcss.config.mjs`, `tailwind.config.ts`: Эти файлы предназначены для настройки Tailwind CSS.
*   `tsconfig.json`: Этот файл предназначен для настройки TypeScript.
*   `README.md`: Этот файл содержит документацию по проекту.

## Файлы в каталоге `src/app`

*   `favicon.ico`: Это иконка, которая отображается на вкладке браузера.
*   `globals.css`: Этот файл предназначен для глобальных стилей CSS.
*   `layout.tsx`: Это основной макет приложения.
*   `page.tsx`: Это главная страница приложения.
